## PASOS DEL DESARROLLO DEL PROYECTO 

1 TENER DESCARGADO NODE JS

2 CREAMOS LA CARPETA ALMA & COCINA

3 ABRIMOS LA CARPETA EN EL VISUAL

4 Y EN LA TERMINAL PONEMOS "npm create vite@latest"

5 LUEGO EN LA TERMINAL PONEMOS npm install

6 Y PARA PODER PRENDER NUESTRO PROYECTO PONEMOS npm run dev
